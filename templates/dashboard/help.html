🚀 Quick Start Guide

1

Set your theme

Toggle dark/light mode using the moon icon in the top right. Advanced users can select custom mesh gradients in settings.

2

Capture a thought

Use the "Brain Dump" sticky note widget on the dashboard to instantly save thoughts. It auto-saves as you type.

3

Trigger Voice Commands

Press Ctrl + Shift + V anywhere to activate the Voice Assistant overlay and control the dashboard hands-free.

4

Review daily analytics

Check the Activity Heatmap on the V7 Dashboard. Consistency builds your daily streak and levels up your profile.
